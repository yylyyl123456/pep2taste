# Bitter Virtual Digestion Prediction Outputs

This directory is generated by `predict_virtual_digest_fasta_combo037.py` using COMBO_037.

Folder layout:

- `predictions/`: raw model prediction table, predicted bitter hits, skipped records, and top candidates.
- `tables/`: summary tables and web-server manifests.
- `download_files/by_hydrolysis_method/`: per-enzyme CSV.GZ files for probability tiers plus Reported/No reported groups.
- `high_confidence_bitter/`: candidates above the configurable high-confidence threshold, including FASTA files for MEME.

The model is run once on `data/unreported_fragments_all.fasta`. The nine hydrolysis-method source files are then used to annotate peptide origin, so method-specific downloads do not require repeating model inference.
