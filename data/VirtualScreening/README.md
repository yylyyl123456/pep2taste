Virtual Screening placeholder
=============================

Place future virtual digestion prediction files here.

Recommended subfolders:
- bitter/: Bitter-Fusion prediction pools and filtered candidates.
- umami/: Umami-LoRA prediction pools and filtered candidates.
- motifs/: N-terminal/C-terminal motif summaries and MEME outputs.

The Streamlit Virtual Screening page reads the manifest CSV in this folder first.
